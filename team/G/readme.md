# Джа'рон #4

Пусть Джа'рон поделил каждую бутылку на `K` кусков. С одной стороны, `MK` делится на `N`. С другой стороны, `K` минимально. Значит, `MK` - наименьшее общее кратное чисел `N` и `M`.
